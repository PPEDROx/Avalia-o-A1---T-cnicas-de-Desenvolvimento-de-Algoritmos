# Avaliação-A1---Técnicas-de-Desenvolvimento-de-Algoritmos
Projetos/Tópicos01. Estruturas Condicionais – Verificação de NotaDescrição: Este projeto demonstra o uso de estruturas condicionais ( if, elif, else) para tomar decisões no código. O exemplo prático é a verificação da nota de um aluno para determinar se ele foi aprovado, reprovado ou se está em recuperação.libras Abordados:if: Executa um bloco de código se a condição for verdadeira .elif(else if): Permite testar múltiplas condições .else: Executa um bloco de código se nenhuma das condições anteriores for verdadeira.Operadores de Comparação (por exemplo,$>, <, >=, <=, ==$).02. Estrutura de Repetição – Usando WHILE e Usando FORDescrição: Este módulo explora as estruturas de repetição (ou laços), que permite executar um bloco de código várias vezes. São apresentados exemplos utilizando os dois principais tipos de laços.Shutterstocklibras Abordados:Laço WHILE: Repete um bloco de código enquanto uma condição é verdadeira.Útil quando o número de repetições é desconhecido ou depende de uma condição.Laço FOR: Itera sobre uma sequência de elementos (como itens em uma lista ou uma faixa numérica).Útil quando o número de repetições é conhecido ou para percorrer coleções.03. Listas – Soma de NúmerosDescrição: Este tópico foca em Listas (Arrays), uma das estruturas de dados mais comuns, que armazena uma coleção ordenada de itens. O exemplo prático é a soma de todos os números contidos em uma lista.libras Abordados:Definição de Lista: Criação e elementos.Indexação: Acesso a elementos pelo índice (começando em 0).Iteração: Uso do laço FORpara percorrer todos os elementos da lista.Métodos Básicos: Adicionar ( append), remover, etc.04. Dicionários – Cadastro e ContatoDescrição: Este módulo apresenta os Dicionários (Mapas ou HashMaps), uma estrutura de dados que armazena informações em pares de chave-valor (em vez de um índice numérico). O exemplo é um cadastro simples de contatos .libras Abordados:Estrutura Chave-Valor: Cada valor é acessado através de sua chave única (ex: nome: "Alice").Criação e Acesso: Como criar um dicionário e como recuperar um valor pela sua chave.Adição/Atualização: Como adicionar novos pares ou modificar valores existentes.Iteração: Percorrer chaves, valores ou itens (pares) do dicionário.
